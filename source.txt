GenSpark AI 슬라이드는 2025 년 4 월에 정식 공개된 프레젠테이션 자동화 기능으로, 주제 한 줄 또는 PDF·Word·Excel 같은 자료를 올리면 AI가 구조‧디자인‧시각 자료까지 일괄 생성해 주는 ‘원-스톱’ 슬라이드 제작 도구입니다. 몇 분 안에 전문가 수준의 데크를 완성할 수 있고, 생성 결과를 브라우저에서 바로 수정하거나 PPT·PDF로 내려받을 수 있어 밤샘 작업을 획기적으로 줄여 줍니다.

## 1. 기능 한눈에 보기

| 주요 기능             | 설명                                              |
| ----------------- | ----------------------------------------------- |
| **자동 슬라이드 생성**    | 주제 입력 또는 문서 업로드만으로 목차-본문-디자인까지 완성               |
| **자연어 편집**        | “7 페이지를 그래프로 바꿔줘”처럼 대화로 레이아웃·차트·동영상 삽입 가능       |
| **리치 에디터**        | 생성된 슬라이드를 실시간 WYSIWYG 방식으로 다시 손질                |
| **멀티미디어 통합**      | 이미지·그래프·영상·오디오를 AI가 자동 삽입 및 정렬                  |
| **딥 리서치**         | 웹·문헌 데이터를 에이전트가 조사해 핵심 통계·인사이트를 슬라이드에 삽입        |
| **팩트체크 & 슬라이드 쇼** | ‘내용 사실 확인’ 버튼과 자동 슬라이드 쇼 재생 지원                  |
| **내보내기 & 협업**     | PPT, PDF, Figma import 등 다채로운 포맷 지원             |
| **크레딧 기반 과금**     | 무료 200 크레딧/일, 3 p 슬라이드 ≈ 230 크레딧 소모—복잡할수록 비용 증가 |

## 2. 시작 가이드

### ① 계정 만들기 & 크레딧 확인

구글 계정으로 가입하면 즉시 200 크레딧이 지급됩니다.

### ② AI Slides 실행

좌측 메뉴 **AI Slides** 선택 → 주제 입력 또는 파일 업로드.

### ③ 초안 점검

AI가 목차·슬라이드 수·디자인 시안을 제안합니다. 마음에 들지 않으면 “페이지 수 12개로 줄여줘” 같이 수정 프롬프트를 보내세요.

### ④ 세부 편집

* 리치 에디터에서 폰트·컬러·레이아웃 직접 수정
* **Add media** 버튼으로 그래프·GIF·영상 삽입
* **Fact-Check** 버튼으로 인용 자료 검증 (크레딧 추가 소모)

### ⑤ 내보내기

완성본은 PPT, PDF, Figma 파일로 저장하거나 웹 링크로 팀과 공유합니다.

## 3. 실전 팁

### 프롬프트 설계

1. **목차 먼저 제시**: “①문제 정의 ②시장규모 ③솔루션…”처럼 구조를 지정하면 콘텐츠 품질이 높아집니다.
2. **데이터 근거 요구**: “각 슬라이드에 최신 통계와 출처 추가”라고 명시하면 리서치 결과가 자동 삽입됩니다.

### 크레딧 절약

* 자료를 한꺼번에 올려 한 번에 생성 → 반복 생성 횟수 최소화.
* 사실 확인·대용량 미디어 삽입은 필요한 페이지만 실행.

### 완성도 높이기

* PPT로 내보낸 뒤 슬라이드 마스터를 활용해 브랜드 컬러·폰트를 일괄 적용.
* 복잡한 차트는 Figma import 후 벡터 객체로 재편집.

## 5. 한계 & 주의사항

* **UI 개선 필요**: 편집 창이 작아 세밀 수정이 불편.
* **PPT 재호환성**: 내보낸 PPT를 다시 불러오면 일부 레이아웃이 깨질 수 있음.
* **할루시네이션**: 일부 그래프나 수치가 부정확하게 생성될 수 있으므로 검증 필수.
* **데이터 기밀**: 업로드 자료는 클라우드 처리되므로 민감 정보는 가급적 삭제 후 사용.

## 6. 앞으로의 로드맵

GenSpark 측은 UI 전체화면 편집, 스마트 템플릿, 크레딧 효율화 등 추가 업데이트를 예고하고 있습니다.  2월 시리즈 A 100 만 달러 투자 유치로 확보한 자본으로 R\&D 속도가 빨라질 전망이니 기능 확장을 주목해 보세요.
